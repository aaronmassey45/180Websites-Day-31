Image Search Abstraction Layer

Free Code Camp: API and Microservices Project

Technologies Used: NodeJS, Express, Javascript, HTML, CSS, and Bootstrap

To search images, enter in the following URL and change 'SEARCHTERM' to what you want to search

http://localhost:3000/api/imagesearch/SEARCHTERM?offset=1
